## ⚙️ Обробка даних

### Data Dictionary

| Змінна | Опис | Тип |
|---|---|---|
| `Price_main` | Загальна вартість об'єкта (USD) | Числовий |
| `area` | Загальна площа квартири (м²) | Числовий |
| `building_age` | Вік будинку (кількість років) | Числовий |
| `type_build` | Категорія житлового фонду | Категоріальний |
| `District` | Адміністративний район | Категоріальний |

### Кроки обробки

1. **Оптимізація** — зміна типів даних, видалення нерелевантних стовпців.
2. **Feature Engineering** — класифікація житлового фонду за функцією `type_build`:
   - `New building` — до 5 років
   - `New housing stock` — від 5 до 25 років
   - `Old housing stock` — понад 25 років
3. **Фільтрація викидів** — видалення аномальних значень ціни та площі.

```python
def type_build(building_age):
    if building_age < 5:
        return 'New building'
    elif building_age <= 25:
        return 'New housing stock'
    else:
        return 'Old housing stock'

# Фільтрація викидів
df_no_outliers = df_clear[
    (df_clear['Price_main'] >= 10000) & 
    (df_clear['Price_main'] <= 2000000) & 
    (df_clear['area'] >= 15)
]
```

## 💡 Ключові інсайти

**1. Районні відмінності у ціні**
Печерський район має найвищу медіанну ціну за м² — понад $3 000/м², тоді як Деснянський залишається найдоступнішим — близько $1 200/м². Різниця між преміальним та економ-сегментом складає майже 2,5× в межах одного міста.

**2. Залежність площі від ціни**
Scatter plot демонструє чітку лінійну залежність між загальною площею та вартістю об'єкта. Водночас у преміум-сегменті (площа >100 м²) дисперсія цін значно зростає — локація та поверх суттєво впливають на підсумкову вартість.

## 📝 Висновки

Ринок нерухомості Києва демонструє виражену сегментацію, де ключовими драйверами ціноутворення виступають географія (район) та площа об'єкта. Найсильнішим предиктором ціни є локація — різниця медіанної вартості м² між Печерським та Деснянським районами сягає 2,5×. Загальна площа лінійно корелює з вартістю, проте у сегменті великогабаритного житла (понад 100 м²) спостерігається значна дисперсія цін. У цьому преміальному сегменті визначальними стають додаткові характеристики об'єкта, які можна детальніше дослідити за допомогою інтерактивних фільтрів дашборда (поверх, рік будівництва).
